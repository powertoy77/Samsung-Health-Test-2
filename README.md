# Samsung Health Clone

삼성 헬스 앱의 UI를 Flutter로 구현한 클론 프로젝트입니다.

## 📱 프로젝트 개요

이 프로젝트는 삼성 헬스 앱의 메인 화면을 Flutter로 구현한 것입니다. 실제 삼성 헬스 앱과 동일한 디자인과 레이아웃을 제공하며, **Daily Activity Detail Page**, **Sleep Detail Page**, **Workout Start Page**, **Energy Score Detail Page**, **Sleep Coaching Start Page**, **All Workouts Page**, **Steps Detail Page** 기능이 포함되어 있습니다.

## ✨ 주요 기능

### 🏠 메인 화면 구성
- **헤더 섹션**: Samsung Health 로고 및 연결 상태 표시
- **활동 요약 카드**: 걸음 수, 활동 시간, 소모 칼로리 표시 ⭐ **탭 가능**
- **수면 점수 카드**: 수면 점수 및 수면 시간 정보 ⭐ **탭 가능**
- **활동 바로가기 카드**: 걷기, 수영, 달리기, 더보기 버튼 ⭐ **탭 가능**
- **에너지 점수 카드**: 에너지 점수 및 구름 캐릭터 ⭐ **탭 가능**
- **수면 코칭 카드**: 수면 동물 유형 확인 안내 ⭐ **탭 가능**
- **주간 운동 기록**: 운동 시작 유도 메시지 ⭐ **탭 가능**
- **걸음 목표 진행률**: 목표 대비 현재 진행률 표시 ⭐ **탭 가능**
- **하단 네비게이션**: 홈, 투게더, Discover, 피트니스, 내 페이지

### 🎯 Daily Activity Detail Page (신규 기능)
**메인 화면의 활동 요약 카드를 탭하면 이동하는 상세 화면**

#### 📊 주요 구성 요소:
1. **날짜 선택기**: 좌우 화살표 또는 캘린더로 날짜 변경
2. **원형 진행률 표시기**: 3겹의 동심원으로 활동 목표 진행률 시각화
   - 가장 바깥쪽: 걸음 수 (967/6,000) - 초록색
   - 중간: 활동 시간 (10/90분) - 파란색
   - 가장 안쪽: 활동 칼로리 (38/500kcal) - 분홍색
3. **활동 지표**: 아이콘과 함께 현재 값/목표 값 표시
4. **요약 정보**: 총 칼로리 소모량 (976 kcal), 이동 거리 (0.76 km)
5. **활동 상세**: 각 항목별 진행률 바
6. **주간 기록**: 일주일간 걸음 수 막대 그래프

#### 🎨 디자인 특징:
- **Material Design 3** 적용
- **원형 진행률 표시기** (percent_indicator 패키지)
- **색상 코딩**: 걸음 수(초록), 시간(파란), 칼로리(분홍)
- **그림자 효과**와 **둥근 모서리**
- **반응형 레이아웃**

### 😴 Sleep Detail Page (신규 기능)

7. **수면 코칭**: 사슴 아이콘과 코칭 재진행 안내
8. **연관 콘텐츠**: CALM, BLESSLIFE 등 4개 콘텐츠

6. **지난밤 수면 상세**: 심박수, 심박변이도, 피부 온도, 호흡수
7. **성취도**: 불꽃 모양 골드 배지 및 성취 표시

- **날짜 선택기**: 8월 17일~24일 에너지 점수 트렌드
- **메인 점수**: 86점 (매우 좋음) +21 상승 트렌드
- **활기찬 생활의 비결**: 건강 상태 분석 및 맞춤형 조언
- **피드백**: 정보 유용성 평가 시스템
- **영향 요인**: 8가지 요인별 진행률 (수면 시간, 규칙성, 심박수 등)
- **수면 상세**: 심박수 58bpm, 심박변이도 47ms, 피부 온도 변화, 호흡수 9.6회/분
- **성취도**: 불꽃 모양 골드 배지 및 "에너지 점수 매우 좋음" 성취

### Sleep Coaching Start Page (신규)
- **동물 유형**: 고슴도치 🦔, 펭귄 🐧, 사자 🦁 3가지 수면 패턴
- **프로그램 안내**: 7일간 수면 기록 요구사항 및 웨어러블 기기 연동
- **코칭 기간**: 3-4주 맞춤형 수면 개선 프로그램
- **시작 액션**: 다음 버튼으로 수면 코칭 시작

### All Workouts Page (신규)
- **상단 네비게이션**: 뒤로가기, "모든 운동" 제목, 필터/정렬, 트렌드, 캘린더, 더보기
- **날짜 선택기**: 일~토요일 가로 스크롤, 8/24 선택 상태
- **이번 주**: 00:00 운동 시간, "더 이상 기록되는 운동이 없습니다" 메시지
- **주간 기록**: 3주간 상세 운동 기록 (8월 17일~23일, 10일~16일, 3일~9일)
- **일별 운동**: 걷기(자동), 리프팅, 수영, 자전거, 기타 운동 등 다양한 운동 유형
- **운동 상세**: 시간, 거리, 시작 시간, 칼로리 소모량 표시

### Steps Detail Page (신규)
- **상단 네비게이션**: 뒤로가기, "걸음수" 제목, 공유/차트/더보기 메뉴
- **날짜 선택기**: 8월 17일~24일 세로 막대 그래프, 8/24 선택 상태
- **메인 걸음수**: 8,234/10,000 (82% 진행률), 6.2km 거리, 387kcal 칼로리, 12층
- **시간별 차트**: 24시간 걸음수 막대 그래프 (오전 6시~오후 6시 피크 시간대)
- **걸음수 비교**: 나이대별 평균 8,500걸음 (97% 달성), 전체 평균 7,200걸음 (114% 달성)
- **배지 시스템**: 골드 배지 "걸음수 마스터" 성취, 일일 목표 달성 표시
- **연관 콘텐츠**: 건강 관련 콘텐츠 4개 (운동 팁, 건강 정보, 영양 가이드, 스트레칭)
