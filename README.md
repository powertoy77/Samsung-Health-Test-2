# Samsung Health Clone

삼성 헬스 앱의 UI를 Flutter로 구현한 클론 프로젝트입니다.

## 📱 프로젝트 개요

이 프로젝트는 삼성 헬스 앱의 메인 화면을 Flutter로 구현한 것입니다. 실제 삼성 헬스 앱과 동일한 디자인과 레이아웃을 제공하며, **Daily Activity Detail Page**, **Sleep Detail Page**, **Workout Start Page**, **Energy Score Detail Page** 기능이 포함되어 있습니다.

## ✨ 주요 기능

### 🏠 메인 화면 구성
- **헤더 섹션**: Samsung Health 로고 및 연결 상태 표시
- **활동 요약 카드**: 걸음 수, 활동 시간, 소모 칼로리 표시 ⭐ **탭 가능**
- **수면 점수 카드**: 수면 점수 및 수면 시간 정보 ⭐ **탭 가능**
- **활동 바로가기 카드**: 걷기, 수영, 달리기, 더보기 버튼 ⭐ **탭 가능**
- **에너지 점수 카드**: 에너지 점수 및 구름 캐릭터 ⭐ **탭 가능**
- **수면 코칭 카드**: 수면 동물 유형 확인 안내
- **주간 운동 기록**: 운동 시작 유도 메시지
- **걸음 목표 진행률**: 목표 대비 현재 진행률 표시
- **하단 네비게이션**: 홈, 투게더, Discover, 피트니스, 내 페이지

### 🎯 Daily Activity Detail Page (신규 기능)
**메인 화면의 활동 요약 카드를 탭하면 이동하는 상세 화면**

#### 📊 주요 구성 요소:
1. **날짜 선택기**: 좌우 화살표 또는 캘린더로 날짜 변경
2. **원형 진행률 표시기**: 3겹의 동심원으로 활동 목표 진행률 시각화
   - 가장 바깥쪽: 걸음 수 (967/6,000) - 초록색
   - 중간: 활동 시간 (10/90분) - 파란색
   - 가장 안쪽: 활동 칼로리 (38/500kcal) - 분홍색
3. **활동 지표**: 아이콘과 함께 현재 값/목표 값 표시
4. **요약 정보**: 총 칼로리 소모량 (976 kcal), 이동 거리 (0.76 km)
5. **활동 상세**: 각 항목별 진행률 바
6. **주간 기록**: 일주일간 걸음 수 막대 그래프

#### 🎨 디자인 특징:
- **Material Design 3** 적용
- **원형 진행률 표시기** (percent_indicator 패키지)
- **색상 코딩**: 걸음 수(초록), 시간(파란), 칼로리(분홍)
- **그림자 효과**와 **둥근 모서리**
- **반응형 레이아웃**

### 😴 Sleep Detail Page (신규 기능)
**메인 화면의 수면 점수 카드를 탭하면 이동하는 상세 화면**

#### 📊 주요 구성 요소:
1. **주간 수면 시간 차트**: 6월 17일~24일 수면 시간 막대 그래프
2. **수면 시간 카드**: 총 수면 시간 (7시간 33분), 실제 수면 시간 (6시간 24분)
3. **수면 점수 카드**: 78.13점 (좋음) 및 영향 요인 분석
4. **수면 단계 카드**: 각성 25%, 얕은 수면 33%, 깊은 수면 39%, 렘 수면 16%
5. **생체 신호 카드**: 혈중 산소, 코골이, 피부 온도, 심박수, 호흡수
6. **수면 규칙성**: 7일간 수면/기상 목표 달성 현황
7. **수면 코칭**: 사슴 아이콘과 코칭 재진행 안내
8. **연관 콘텐츠**: CALM, BLESSLIFE 등 4개 콘텐츠
9. **수면 기록 추가**: 하단 고정 버튼

#### 🎨 디자인 특징:
- **Material Design 3** 적용
- **진행률 바** (percent_indicator 패키지)
- **색상 코딩**: 각 수면 단계별 색상 구분
- **그림자 효과**와 **둥근 모서리**
- **반응형 레이아웃**

### 🏃‍♂️ Workout Start Page (신규 기능)
**메인 화면의 활동 바로가기 카드를 탭하면 이동하는 운동 시작 화면**

#### 📊 주요 구성 요소:
1. **운동 카테고리**: 6가지 운동 유형 (걷기, 달리기, 자전거, 수영, 요가, 헬스)
2. **인기 운동**: 빠른 걷기, 조깅, 자전거 타기 (시간 및 칼로리 정보)
3. **최근 운동**: 오늘 걷기 (7,234걸음, 45분), 어제 조깅 (5.2km, 32분)
4. **이번 주 목표**: 운동 시간 (3/5일, 60%), 총 운동 시간 (2시간 15분, 45%)

#### 🎨 디자인 특징:
- **Material Design 3** 적용
- **그리드 레이아웃**으로 운동 카테고리 표시
- **색상 코딩**: 각 운동별 고유 색상 (걷기-초록, 달리기-주황, 자전거-파랑 등)
- **진행률 바** (percent_indicator 패키지)
- **그림자 효과**와 **둥근 모서리**
- **반응형 레이아웃**

### ⚡ Energy Score Detail Page (신규 기능)
**메인 화면의 에너지 점수 카드를 탭하면 이동하는 에너지 점수 상세 화면**

#### 📊 주요 구성 요소:
1. **날짜 선택기**: 8월 17일~24일 에너지 점수 트렌드
2. **메인 에너지 점수**: 반원형 진행률 표시기 (86점, 86% 진행률)
3. **활기찬 생활의 비결**: 건강 상태 분석 및 맞춤형 조언
4. **피드백 섹션**: 정보 유용성 평가 (좋아요/싫어요)
5. **에너지 점수 영향 요인**: 8가지 요인별 진행률 바
   - 수면 시간 평균, 규칙성, 패턴, 입면 시간
   - 전날 활동, 활동 규칙성, 수면 심박수, 심박변이도
6. **지난밤 수면 상세**: 심박수, 심박변이도, 피부 온도, 호흡수
7. **성취도**: 불꽃 모양 골드 배지 및 성취 표시

#### 🎨 디자인 특징:
- **Material Design 3** 적용
- **반원형 진행률 표시기** (percent_indicator 패키지)
- **색상 코딩**: 파란색(매우 좋음), 회색(좋음)
- **그림자 효과**와 **둥근 모서리**
- **반응형 레이아웃**과 **스크롤 가능한 콘텐츠**

### 🎨 디자인 특징
