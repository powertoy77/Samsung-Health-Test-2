## 📱 2. 상세 페이지 메뉴 요구사항

### 2.1 일일 활동 상세 페이지 (Daily Activity Detail)

#### 2.1.1 페이지 접근 방법
- **진입점**: 메인 화면의 "활동 요약 카드" 탭
- **카드 정보**: 713 걸음, 7분, 27 kcal 표시

#### 2.1.2 메뉴 구성 요소

##### 📅 날짜 선택 메뉴
- **기능**: 좌우 화살표 또는 캘린더로 날짜 변경
- **표시 형식**: "8월 24일" 형태
- **동작**: 
  - 좌측 화살표: 이전 날짜로 이동
  - 우측 화살표: 다음 날짜로 이동
  - 날짜 텍스트 탭: 캘린더 모달 열기

##### 📊 원형 진행률 표시기
- **구조**: 3겹의 동심원
- **데이터 표시**:
  - 가장 바깥쪽: 걸음 수 (967/6,000) - 초록색
  - 중간: 활동 시간 (10/90분) - 파란색
  - 가장 안쪽: 활동 칼로리 (38/500kcal) - 분홍색
- **시각적 효과**: 각 원의 진행률에 따른 색상 채움

##### 📈 활동 지표 메뉴
- **구성 요소**:
  - 걸음 수: 아이콘 + "967 / 6,000" + "걸음"
  - 활동 시간: 아이콘 + "10 / 90" + "분"
  - 활동 칼로리: 아이콘 + "38 / 500" + "kcal"
- **레이아웃**: 가로 배치, 각 항목별 카드 형태

##### 💡 요약 정보 메뉴
- **총 칼로리 소모량**: "976 kcal"
- **이동 거리**: "0.76 km"
- **표시 형식**: 큰 숫자 + 단위

##### 📋 활동 상세 메뉴
- **구성**: 각 항목별 진행률 바
- **데이터**:
  - 걸음 수 진행률: 16.1%
  - 활동 시간 진행률: 11.1%
  - 활동 칼로리 진행률: 7.6%
- **시각적 표현**: 선형 진행률 바 + 퍼센트 표시

##### 📊 주간 기록 메뉴
- **형태**: 일주일간 걸음 수 막대 그래프
- **데이터 범위**: 월~일요일
- **인터랙션**: 각 막대 탭 시 해당 날짜로 이동

##### ⏰ 수면 시간 카드

##### 📈 수면 점수 카드
- **점수**: "78.13점 (좋음)"
- **영향 요인**: 
  - 수면 시간
  - 수면 규칙성
  - 수면 효율성
- **시각적 표현**: 원형 진행률 표시기

  - 깊은 수면: 39% (진한 파란색)
  - 렘 수면: 16% (보라색)

##### 💓 생체 신호 카드

- **코골이**: 감지 여부

- **심박수**: "58 bpm"
- **호흡수**: "9.6 회/분"

##### 📅 수면 규칙성
- **기간**: 7일간 수면/기상 목표 달성 현황
- **표시**: 달성일/전체일 형태
- **시각적 표현**: 체크마크 또는 X 표시

##### 🦌 수면 코칭
- **아이콘**: 사슴 아이콘
- **텍스트**: "코칭 재진행"

- **인터랙션**: 각 콘텐츠 탭 시 상세 페이지

##### ➕ 수면 기록 추가

### 2.3 운동 시작 페이지 (Workout Start)

#### 2.3.1 페이지 접근 방법
- **진입점**: 메인 화면의 "활동 바로가기 카드" 탭
- **카드 정보**: 걷기, 수영, 달리기, 더보기 버튼

#### 2.3.2 메뉴 구성 요소

##### 🏃‍♂️ 운동 카테고리 메뉴
- **구성**: 6가지 운동 유형
- **레이아웃**: 3x2 그리드
- **운동 종류**:
  - 걷기 (초록색)
  - 달리기 (주황색)
  - 자전거 (파란색)
  - 수영 (청록색)
  - 요가 (보라색)
  - 헬스 (회색)
- **인터랙션**: 각 카테고리 탭 시 해당 운동 시작

##### 🔥 인기 운동 메뉴
- **빠른 걷기**: 30분, 150kcal
- **조깅**: 45분, 300kcal
- **자전거 타기**: 60분, 400kcal
- **레이아웃**: 가로 스크롤 카드
- **액션**: 각 운동 탭 시 즉시 시작

##### 📝 최근 운동 메뉴
- **오늘 걷기**: 7,234걸음, 45분
- **어제 조깅**: 5.2km, 32분
- **표시 형식**: 운동 아이콘 + 데이터 + 시간
- **액션**: 탭 시 해당 운동 상세 정보

##### 🎯 이번 주 목표 메뉴
- **운동 시간**: 3/5일, 60% 진행률
- **총 운동 시간**: 2시간 15분, 45% 진행률
- **시각적 표현**: 진행률 바 + 퍼센트
- **색상**: 초록색 (달성), 회색 (미달성)
