## 🎯 1. 제품 개요

### 1.1 제품 정의
Samsung Health 앱의 UI를 Flutter로 구현한 클론 프로젝트로, 실제 삼성 헬스 앱과 동일한 디자인과 레이아웃을 제공하는 건강 관리 애플리케이션입니다.

### 1.2 제품 목표
- Samsung Health 앱의 핵심 기능들을 Flutter로 재현
- Material Design 3 기반의 현대적 UI/UX 제공
- 실제 Samsung Health Data SDK 연동을 통한 데이터 처리
- 교육 및 학습 목적의 완전한 기능 구현

### 1.3 타겟 사용자
- Flutter 개발 학습자
- UI/UX 디자인 연구자
- Samsung Health 앱 기능 분석자
- 모바일 앱 개발 교육자

## 📱 3. 기능 요구사항

### 3.1 핵심 기능 목록

#### 3.1.1 메인 화면 (홈)
- **헤더 섹션**: Samsung Health 로고, 연결 상태, 권한 요청 버튼
- **활동 요약 카드**: 걸음 수, 활동 시간, 소모 칼로리 표시 (탭 가능)
- **수면 점수 카드**: 수면 점수 및 수면 시간 정보 (탭 가능)
- **활동 바로가기 카드**: 걷기, 수영, 달리기, 더보기 버튼 (탭 가능)
- **에너지 점수 카드**: 에너지 점수 및 구름 캐릭터 (탭 가능)
- **수면 코칭 카드**: 수면 동물 유형 확인 안내 (탭 가능)
- **주간 운동 기록**: 운동 시작 유도 메시지 (탭 가능)
- **걸음 목표 진행률**: 목표 대비 현재 진행률 표시 (탭 가능)
- **하단 네비게이션**: 홈, 투게더, 발견, 피트니스, 내 페이지

#### 3.1.2 상세 페이지들
1. **일일 활동 상세 페이지**
   - 날짜 선택기
   - 원형 진행률 표시기 (3겹 동심원)
   - 활동 지표 및 요약 정보
   - 주간 기록 그래프

2. **수면 상세 페이지**
   - 주간 수면 시간 차트
   - 수면 시간 및 점수 카드
   - 수면 단계 분석
   - 생체 신호 정보

3. **운동 시작 페이지**
   - 6가지 운동 카테고리
   - 인기 운동 및 최근 운동
   - 주간 목표 진행률

4. **에너지 점수 상세 페이지**
   - 날짜 선택기 및 트렌드
   - 반원형 진행률 표시기
   - 8가지 영향 요인 분석
   - 성취도 및 배지

5. **수면 코칭 시작 페이지**
   - 3가지 동물 유형 카드
   - 프로그램 안내
   - 시작 액션

6. **모든 운동 페이지**
   - 상단 네비게이션
   - 날짜 선택기
   - 주간 운동 기록 (3주간)
   - 일별 상세 운동 기록

7. **걸음수 상세 페이지**
   - 날짜별 걸음수 그래프
   - 메인 걸음수 카드
   - 시간별 걸음수 차트
   - 배지 시스템

#### 3.1.3 하단 네비게이션 탭들
1. **투게더 페이지**
   - 사용자 프로필 정보
   - 걸음 순위판
   - 챌린지 카드들

2. **Discover 페이지**
   - Health Hub 온보딩 카드
   - 연결 네트워크 시각화
   - Health Hub 아이콘들

3. **피트니스 페이지**
   - 운동 콘텐츠 섹션들
   - Fitness Programs
   - Partner Services
   - By Provider

4. **내 페이지**
   - 프로필 섹션
   - 헬스 데이터 공유
   - 주별 분석
   - 배지, 개인 최고기록, 도전

### 3.2 Samsung Health SDK 연동 기능

#### 3.2.1 권한 관리 시스템
- **지원 권한**:
  - 걸음수 읽기 권한 (`com.samsung.shealth.step_count`)
  - 수면 데이터 읽기 권한 (`com.samsung.shealth.sleep`)
  - 체중 데이터 쓰기 권한 (`com.samsung.shealth.weight`)

- **주요 기능**:
  - 개별/전체 권한 요청
  - 실시간 권한 상태 확인
  - 권한 거부 시 가이드 다이얼로그
  - 설정 화면 이동 기능

#### 3.2.2 데이터 처리 시스템
- **HealthStore 연결 관리**
- **HealthDataResolver를 통한 데이터 읽기**
- **Kotlin Coroutines를 활용한 비동기 처리**
- **MethodChannel을 통한 Flutter-Android 통신**
